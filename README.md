# LaGuitar API RESTful (Backend)

Esta es la API desarrollada en Node.js, Express y PostgreSQL para el proyecto "LaGuitar". Este servidor actúa como backend para proveer y gestionar los datos del catálogo de guitarras consumidos por el frontend en React.

## Requisitos Previos
- Node.js instalado.
- Una base de datos PostgreSQL (puede ser local o en la nube mediante servicios como Render).
